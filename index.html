La Web Moderna

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y organizar el contenido de una página web, como texto, imágenes, y enlaces. Utiliza etiquetas para definir elementos que se muestran en los navegadores. Es el pilar básico del desarrollo web junto con CSS y JavaScript.

Fuentes HTML

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para diseñar y dar estilo a las páginas web, como colores, fuentes, y distribución del contenido. Permite separar el diseño de la estructura HTML para un desarrollo más organizado. Es clave para crear sitios web atractivos y responsivos.

Fuentes CSS

JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad y funcionalidades dinámicas a las páginas web, como animaciones, validaciones de formularios o actualizaciones en tiempo real. Se ejecuta en los navegadores y también en servidores con tecnologías como Node.js. Es esencial en el desarrollo web junto con HTML y CSS.

Fuentes JavaScript

HTTP

HTTP (HyperText Transfer Protocol) es un protocolo de comunicación que permite la transferencia de información entre un cliente (como un navegador) y un servidor web. Es el fundamento de cómo se cargan páginas, enviando solicitudes y recibiendo respuestas. Funciona con métodos como GET para obtener datos y POST para enviarlos.

Fuentes HTTP
